PCLoop: User Manual

Prerequisites:
Before running PCLoop, ensure you have the following prerequisites installed:
1. Python 3.12.1
2. pip 23.2.1
3. Django

Open the project in cmd then run command below:
1. pip install shortuuid
2. pip install django-paypal
3. pip install django-jazzmin
4. pip install django-taggit
6. pip install --upgrade django

After all the installation then run command:
1. python manage.py runserver

For create a admin account run command:
1.python manage.py createsuperuser
After account created
1. python manage.py runserver
